# 🤖 AI Chatbot

An interactive AI Chatbot developed using **HTML, CSS, and JavaScript**. The chatbot provides quick responses to common user queries such as attendance, CGPA, interview preparation, placements, library information, and subject details through a simple and responsive user interface.

---

## 🚀 Features

- 🔐 User Login System
- 💬 Interactive Chat Interface
- 🎓 College Information Support
- 📚 Subject Information
- 📊 CGPA Calculator
- 📅 Attendance Calculator
- 💼 Placement Preparation Tips
- 🎤 Interview Questions & Answers
- 📖 Library Information
- 🔍 Fast Response System
- 📱 Responsive Design
- 🎨 Modern User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ▶️ How to Run

1. Download or clone the repository.

```
git clone https://github.com/your-username/AI-Chatbot.git
```

2. Open the project folder.

3. Run **login.html** using Live Server or any modern web browser.

4. Log in and start chatting with the AI chatbot.
